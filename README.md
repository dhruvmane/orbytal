# ORBYTAL
- **Orbytal is a unified, interactive platform for accessing space events, missions and cosmic data.**
- **It provides Real-Time updates in a simple, beginner-friendly format.**
- **Location-based maps and alerts helps users track visible celestial events.**
- **Space Missions are shown via visual timelines for easy exploration and understanding.**
- **Cosmic Weather Insights like Solar Storms and Auroras are simplified for public awareness.**
- **A learning hub explaining real-world uses of satellite data in areas such as climate and disaster management.**

## Tech Stack
- UI/Design: Figma
- Frontend: React.js, JS, HTML5, CSS3, TailwindCSS
- Backend: Flask, Node.js
- Database: MySQL
